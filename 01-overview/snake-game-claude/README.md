# Snake Game (Claude)

This folder contains a small Snake game built with React and Vite. It was developed using Claude for code generation and instructions; the implementation demonstrates how an LLM can bootstrap a playable frontend app with styling, input handling, and game logic.

Run the app locally

```bash
npm install
npm run dev
```
