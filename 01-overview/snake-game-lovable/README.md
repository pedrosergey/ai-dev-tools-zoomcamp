# Snake Game (Lovable)

This folder contains a Snake game project created from scratch using the lovable.dev bootstrapper. It showcases a small app scaffolded by a project generator and adapted for deployment.

Run the app locally

```bash
cd 01-overview/snake-game-lovable
npm install
npm run dev
```
